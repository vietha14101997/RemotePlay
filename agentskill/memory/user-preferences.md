# Sở thích người dùng

## Ngôn ngữ
- **Ngôn ngữ ưu tiên:** Tiếng Việt.
- **Yêu cầu:** Tất cả các giải thích, tài liệu hướng dẫn (trừ code comment nếu không cần thiết) và phản hồi nên được thực hiện bằng tiếng Việt.

## Phong cách lập trình
- Ưu tiên sử dụng các thư viện hiện đại của Android: Jetpack Compose, Kotlin Coroutines, Flow, Hilt.
- Tuân thủ nguyên tắc Clean Architecture và MVVM.
- Code sạch, dễ đọc và có giải thích rõ ràng bằng tiếng Việt khi cần.

## Quy trình làm việc
- Luôn cập nhật bộ nhớ dài hạn (`agentskill/memory/`) khi có quyết định quan trọng hoặc thay đổi lớn về kiến trúc.
- Sử dụng hệ thống Agent để giải quyết các tác vụ phức tạp.
