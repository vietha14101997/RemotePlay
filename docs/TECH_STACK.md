# Tech Stack

Danh sách các công nghệ và thư viện được sử dụng trong dự án Remote Play.

## 📱 Mobile Platform
- **Kotlin**: Ngôn ngữ lập trình chính.
- **Jetpack Compose**: Bộ công cụ hiện đại để xây dựng giao diện người dùng gốc.
- **Android Architecture Components**: ViewModel, LiveData/Flow, Navigation.

## 🌐 Networking & Streaming
- **WebRTC**: Giao thức truyền tải video/audio thời gian thực với độ trễ thấp.
- **WebSocket**: Giao tiếp hai chiều giữa Client và Server (Signaling).
- **Retrofit & OkHttp**: Gọi API RESTful.
- **Protocol Buffers (protobuf)**: Có thể được dùng để tối ưu hóa gói tin điều khiển.

## 💉 Dependency Injection
- **Hilt / Dagger**: Quản lý phụ thuộc.

## 🧵 Concurrency
- **Kotlin Coroutines**: Xử lý bất đồng bộ.
- **Flow**: Luồng dữ liệu phản hồi.

## 🛠 Tools
- **Gradle Kotlin DSL**: Cấu hình build dự án.
- **ClaudeKit (.claude)**: Hệ thống hỗ trợ phát triển bằng AI.
